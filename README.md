# Линейная регрессия

**Линейная регрессия** — это статистический метод, используемый для моделирования взаимосвязи между зависимой переменной и одной или несколькими независимыми переменными.










## Одномерная линейная регрессия

Математическая формулировка одномерной линейной регрессии

$$ y = a * x + b, $$

где:

- $y$ — зависимая переменная
- $x$ — независимая переменная
- $a$ — коэффициент наклона
- $b$ — коэффициент сдвига

Cгенирируем данные и обучим на них линейную модель.

```
from sklearn.datasets import make_regression

X, y = make_regression(n_samples=50, n_features=1, n_informative=1, noise=10, random_state=11)
```

Отобразим данные на графике.

![](/images/1.png "")

Возьмем модель `LinearRegression` из `sklearn` из модуля `linear_model`.

```
from sklearn.linear_model import LinearRegression

model = LinearRegression()
```

И передадим в неё в метод `fit` данные, которые получили выше.

```
model.fit(X, y)
```

Модель обучилась. Обучение линейной модели заключается в поиске коэффициентов, конкретно в нашей задаче - это коэффициент сдвига и наклона.
Вот и получили два коэффициента, осталось их подставить в уравнение прямой и будет готовая линейная модель.

```
model_a = model.coef_[0]
model_b = model.intercept_
```

Отобразим линию, наилучшим образом проходящую вдоль точек.
