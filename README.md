# Метрики классификации

## Accuracy
Что измеряет: Общую долю правильных предсказаний модели среди всех наблюдений.
Формула: '(TP + TN) / (TP + TN + FP + FN)'
## Precision
Что измеряет: Качество положительных предсказаний. Из всех объектов, которые модель назвала положительными (класс 1), какую долю она угадала верно?
Формула: 'TP / (TP + FP)'
## Recall
Что измеряет: Способность модели находить все релевантные объекты (все объекты истинного класса 1). Какую долю от всех реально положительных объектов модель обнаружила?
Формула: 'TP / (TP + FN)'
## F1-score
Что измеряет: Гармоническое среднее между Precision и Recall. Попытка объединить две эти метрики в одну.
Формула: 2 * (Precision * Recall) / (Precision + Recall)
